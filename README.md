# United Airlines Departure Delays and Gains

> This is two separate but related projects. The first project examines how weather impacts departure delays. The second project examine gain (the time made up in the air).

---

## Project Structure

```
├── data/                 # Raw and processed data
├── code/                 # R markdown
├── reports/              # Generated reports and visualizations
└── README.md             # Project documentation
```
